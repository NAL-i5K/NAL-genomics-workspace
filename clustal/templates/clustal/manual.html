Clustal Omega / ClustalW : Multiple alignment of nucleic acid and protein sequences

Reference: http://www.clustal.org/

Introduction

ClustalW is the oldest of the currently most widely used programs for multiple sequence alignment. Clustal Omega is the latest version of CLUSTAL series. ClustalO is faster and more accuracy because of new HMM alignment engine.


i5K-Clustal Tutorial

In this tutorial, we explain some of the features of the CLUSTAL site.

To begin, navigate to the HMMER site via the 'Web Apps->CLUSTAL' menu bar, or go directly to http://i5k.nal.usda.gov/clustal/.

We implemented ClustalW and Clustal Omega in website; they both do multiple sequence alignment just in different ways, users can choose either one based on preference. But two programs have totally different parameters setting, please reference manuals for details.

ClustalW Manual: http://www.clustal.org/download/clustalw_help.txt



Clustal Omega Manual: http://www.clustal.org/omega/README



Paste your query sequence into the 'Query Sequence' field, or load it from disk.

CLUSTAL program only accepts fasta file as query input. Once the correct CLUSTAL program is selected, choosing parameters which are suitable for individual condition. Hit the 'Search' button when finished.



A page will display indicating that your query is running. You can store the indicated query link if you wish to come back to your CLUSTAL results at a later time. Results are stored for one week.



When the CLUSTAL program has finished, your results will be displayed.

In 'Download', you could download alignment file (showed in 'Details Report', phylogenetic tree (if available) and submission details including parameters used in this job.

In 'Details Reports', 'Colorful' button makes alignment colorful for visualization and and 'To Hmmsearch' button redirects users to hmmsearch using the result as input of hmmsearch.



If having user account of I5K workspace, you can review your jobs by clicking 'Queries' tab.
